# Smart Notes Demo 数据说明

## 目标

这套数据不是《卡片笔记写作法》的书摘库，而是用原创、人话笔记展示研思录当前主流程：

```text
随笔 / 文献笔记
  -> 永久笔记
  -> 有理由的关联
  -> 围绕问题的主题索引
  -> 写作提纲和草稿
```

## 当前内容

- 1 条方法来源说明
- 4 条随笔，包含已处理和待处理状态
- 9 条文献笔记，包含已转换和待处理状态
- 48 条永久笔记
- 50 条人工关系，覆盖相关、同主题、支持、补充、反驳、限定、例子、前提和桥接
- 54 条由正文链接自动识别的关系；共 104 条关系
- 18 个主题索引
- 2 个写作项目和对应提纲
- 3 篇示例帮助文章
- 12 条简短导览笔记

## 内容范围

Demo 只讲三件事：

1. 卡片笔记写作法的核心实践。
2. 研思录当前已经实现的操作路径。
3. AI 如何在具体任务旁提供候选帮助。

不放入与主路径无关的主题，也不复刻原书内容。

## AI 边界

AI 可用于材料提炼、候选关联、提纲和草稿检查。所有结果必须先让用户看见，可编辑、可忽略、可确认；不自动创建正式笔记、关系或文章。

## 维护方式

唯一数据源：

```text
scripts/smart-notes-demo-data.mjs
```

刷新 fixture：

```text
npm run demo:refresh
```

生成文件：

```text
tests/fixtures/demo-smart-notes-product-thinking/demo.json
```

不要手工维护多套 Demo，也不要把测试 Demo 混入真实用户 vault。
