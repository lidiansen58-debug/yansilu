# Marketing Site Product Doc

## 1. Document Goal

This document defines the first public marketing website for Yansilu, including:

- brand narrative
- homepage information architecture
- registration and login entry points
- pricing and paid conversion
- billing and account flows

It is intended to align product, design, and implementation before building the site.

## 2. Product Context

### 2.1 Product Name

- Chinese: `研思录`
- English: `Yansilu`
- English descriptor: `Thinking in Notes`

### 2.2 Core Positioning

`研思录不是帮用户记更多，而是帮用户把知识变成自己的智慧。`

`Yansilu is not built to help users save more notes. It is built to help them turn knowledge into their own wisdom.`

### 2.3 Product Nature

Yansilu is not:

- a generic note-taking tool
- a pure AI writing tool
- a simple knowledge archive

Yansilu is:

- a thinking workspace for research and writing
- a local-first note and structure system
- a workflow that connects source material, understanding, compression, indexing, and writing

## 3. Website Goal

The marketing site should do five things:

1. Explain what Yansilu is
2. Make the philosophy memorable
3. Show how the workflow works
4. Convert visitors into registered users
5. Convert serious users into paid subscribers

## 4. Target Audience

### 4.1 Primary Audience

- students and researchers
- knowledge writers
- independent creators building long-form ideas

### 4.2 Shared Pain Points

- they collect too much material without forming stable understanding
- they write many notes without forming reusable viewpoints
- they restart from scratch when writing
- they want AI help, but do not want to lose authorship

## 5. Core Messaging

### 5.1 Main Message

`把知识变成你的智慧。`

### 5.2 Supporting Message

`研思录是一套为研究、写作与长期知识积累设计的思维工作台。它不是替你写作的 AI，而是帮助你把知识、笔记、观点与写作结构连成一条路，并逐步沉淀成你的智慧。`

### 5.3 Key Differentiators

- paraphrase-first note taking
- one-sentence thesis for permanent notes
- wisdom-building through compression and writing
- index cards as thinking structures, not tags
- traceable writing scaffold
- AI with boundaries
