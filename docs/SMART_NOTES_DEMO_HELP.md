# Smart Notes Demo 使用帮助

Smart Notes Demo 用一组可操作的笔记，展示研思录怎样把卡片笔记写作法变成日常动作：

记录 -> 永久笔记 -> 关联 -> 主题 -> 写作

核心不是记得更多，而是**让笔记生长为思想**。

## 第一次体验

1. 从首页处理一条随笔或文献笔记。
2. 打开一条永久笔记，看它怎样写清判断、理由和边界。
3. 给“待关联练习：保存关系前先写清楚为什么”建立一条关联。
4. 打开“为什么要关联笔记？”或“永久笔记是什么？”。
5. 从主题进入写作中心，查看由关键笔记生成的提纲。

一次只做一个动作也可以。

## 三类笔记

- 随笔：先抓住现场想法，之后再处理。
- 文献笔记：保留来源，用自己的话重说材料。
- 永久笔记：写成一条以后还能继续使用、修改或反驳的判断。

## 怎样关联

搜索并选择另一条永久笔记，选择关系类型，写一句为什么相关，然后保存。

- 支持：以后可作为证据。
- 反驳：以后可作为反方。
- 限定：说明判断的适用边界。
- 桥接：连接两个原本分开的主题。
- 例子：把抽象判断放进具体场景。
- 相关：暂时只能确定有关，仍要把理由写清楚。

关联是双向可见的。关系理由比连线数量更重要。

## 怎样形成主题

当几条永久笔记能一起回答一个问题时，可以创建主题索引。先写中心问题，再选择关键笔记和阅读顺序。主题成熟后，可直接进入写作中心。

## 怎样开始写作

写作中心从主题和相关笔记开始：

1. 确认要回答的问题。
2. 选择真正需要的笔记。
3. 生成或手动整理提纲。
4. 检查证据、反方和边界。
5. 保存草稿。

## AI 在哪里帮忙

AI 不增加一条独立流程，只出现在具体任务旁：

- 材料旁：帮我提炼，或提炼选中的一段。
- 关联时：补充少量候选笔记和推荐理由。
- 写作时：生成提纲，检查证据、反方和边界缺口。

所有 AI 结果都只是候选，可编辑、可忽略，保存前必须由用户确认。不配置 AI，也能完整使用研思录。

第一次使用 AI 时，进入“设置 -> AI 设置”。可选择本地 Ollama，或填写 OpenAI 接口兼容服务。

## 手机和电脑怎样配合

手机适合快速记录、拍资料和轻量查看；电脑适合转述、关联、主题和写作。手机保存的内容会进入电脑端首页继续整理。
