# 研思录 UI 原型方案 P0 v1.2

## 1. 原型目标

本原型用于让设计师和研发快速理解 v1 核心交互，不追求视觉细节，优先验证：

1. 5 分钟最小闭环可跑通
2. 复杂度按层解锁，不一次性压给用户
3. 书摘笔记 -> 原创笔记转化足够顺
4. 原创笔记关联与反向链接直观可用
5. 10,000 原创笔记下仍可操作

---

## 2. 信息架构（分层）

## L0 入门层（默认可见）

1. 卡片盒中心（工作台首页）
2. 随笔记录
3. 书摘笔记工坊

## L1 成型层（完成首条书摘后解锁）

1. 原创笔记编辑器

## L2 结构层（完成首条原创后解锁）

1. 索引中心
2. 图谱工作台

## L3 写作层（完成首个索引后解锁）

1. 写作中心

---

## 3. 页面与关键交互

## 3.1 卡片盒中心（工作台首页）

包含：
1. 卡片盒切换（默认：随笔/书摘/原创）
2. 子卡片盒管理（新建、隐藏、删除）
3. 新建卡片盒时设置“最大卡片数量”（默认 500）
4. onboarding 进度条（0/5 -> 5/5）
5. 单一主行动按钮（下一步推荐）

关键反馈：
- 未解锁模块显示“为什么未解锁 + 如何解锁”。
- 每次完成动作后显示“你获得了什么价值”。

## 3.2 随笔记录

包含：
1. 文本输入 + 语音输入
2. 新建标记（未处理）
3. 超过 7 天内容默认隐藏（可筛选显示）
4. “编写原创笔记”快捷入口

关键流程：
- 随笔 -> 原创笔记保存成功 -> 原随笔自动归档。

## 3.3 书摘笔记工坊

包含：
1. quote 与 paraphrase 分区输入
2. 书摘元数据字段：作者/年份/书名/出版社/页码/版本/译者编者
3. 保存校验：页码与转述缺失时阻断
4. 保存后主按钮“新增原创笔记”

## 3.4 原创笔记编辑器（重点）

包含：
1. 结构化字段：核心观点、为什么成立、边界/反例
2. 来源绑定与书摘引用管理
3. originality guard 反馈（pass/warning/blocked）
4. `[[wikilink]]` 输入与候选联想
5. 关系面板（relation_type + rationale）
6. Backlinks 面板（提及/明确关系）
7. Markdown 工具栏（可点按钮，不只键盘输入）
8. 标签面板（标签候选、标签管理、标签结果列表）

关键约束：
1. 不提供“AI 一键生成原创正文”。
2. `blocked` 不能保存 active。
3. `free_link` 缺少 rationale 时阻断提交。

快捷键区（原型必须展示）：
1. 写作：`Ctrl+B`、`Ctrl+I`、`Ctrl+Shift+H`、`Ctrl+\``、`Ctrl+/`、`Ctrl+E`、`Ctrl+R`
2. 结构：`Ctrl+1...6`、`Ctrl+L`、`Alt+Q`
3. 检索：`Ctrl+F`、`Ctrl+Shift+F`、`F3`、`Shift+F3`、`Ctrl+Shift+O`、`Ctrl+Shift+G`
4. Mac 提示：`Ctrl -> Cmd`，`Alt -> Option`

工具栏按钮区（原型必须展示）：
1. 加粗、斜体、高亮、行内代码、公式、引用、分割线
2. `[[关联笔记]]` 按钮（点击即触发候选面板）
3. `#标签` 按钮（点击即触发标签候选面板）
4. 保存按钮

标签交互（原型必须展示）：
1. 点击标签后，进入“该标签结果页”
2. 结果范围默认限制为“当前卡片盒 + 原创笔记”
3. 结果页支持按更新时间排序与子卡片盒筛选

## 3.5 索引中心

包含：
1. 四类索引创建与维护
2. 每个卡片盒独立索引区
3. 主题索引笔记（特殊原创笔记）
4. 索引项排序、理由补充、回跳原笔记

新增提醒：
- 原创笔记净新增每 15 条，触发“整理主题索引”提醒。

## 3.6 图谱工作台

包含：
1. 本地图/主题图/写作图切换
2. 关系过滤、标签过滤、路径高亮
3. 节点详情侧栏与回跳

10k 交互策略：
1. 默认局部图（不默认全局图）
2. 节点阈值折叠与聚类
3. 邻居分层展开（1~3 hops）
4. 联想和搜索 Top N + 分页

## 3.7 写作中心

包含：
1. 写作项目管理
2. 写作篮管理
3. 仅允许“原创卡片盒”笔记进入写作篮
4. 脚手架生成与证据映射

写作引导（八步）：
1. 选主题
2. 设目标
3. 规划研究
4. 收集资料
5. 形成书摘
6. 提炼原创
7. 组写作篮
8. 出脚手架

---

## 4. 5 分钟首轮脚本（演示）

1. 导入样例或新建一条随笔记录
2. 补一条书摘笔记（含转述与页码）
3. 从书摘一键新增原创笔记并保存
4. 建一个主题索引并加入 2~3 条原创笔记
5. 进入写作中心生成首份脚手架

---

## 5. 设计交付要求（给设计师）

每页提供：
1. 正常态、空态、错误态、加载态
2. onboarding 前/中/后状态
3. 未解锁/已解锁对照
4. 中文与英文双语文案占位（走 i18n key）
