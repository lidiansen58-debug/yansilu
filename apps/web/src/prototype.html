切换工作区
浏览本机目录后，新的 Vault 路径会先填进输入框，再由你确认是否切换。
切换后会重新加载目录树，并清空当前打开的编辑标签页；主内容仍以本地 Markdown 为准，SQLite 缓存可重建。
桌面端说明
这一版原型已经为桌面文件选择器、显示文件位置和 Vault 切换预留了接口。你现在可以把它当成真实桌面 App 的设置面板来测试。
  • 浏览目录：优先调用桌面选择器，浏览器环境自动降级为手动输入。
  • 切换 Vault：重新初始化目录树、标签页和缓存上下文。
  • 主内容来源：Markdown 是主内容源，SQLite 负责关系、索引和查询加速。
