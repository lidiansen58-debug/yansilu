    推荐安装流程

    1. 01

      进入下载页或 GitHub Release，选择 Windows 安装包。

    2. 02

      下载安装包，并完成标准安装；如遇 SmartScreen 提醒，请确认来源后继续。

    3. 03

      首次启动后选择或创建本地 Vault。

    4. 04

      登录账户，同步订阅和授权状态。

    5. 05

      后续在“设置 -> 版本更新”里检查新版本。
